Monderman · Model Eval Harness

Runs one fixed fixture set against the live API and downloads a master JSON. Run it once per model (set ANTHROPIC_MODEL on Render, redeploy, then run) so the two files can be diffed. Scores are deterministic and will match across models — the comparison is the narrative prose.
